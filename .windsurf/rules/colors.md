# Paleta de colores

- `#F8F5EC`: Fondo principal
- `#4D7B38`: Fondo de menú de navegación y footer.
- `#FFFFFF`: Texto de menú de navegación y footer.
- `#78A75A`: Menú de navegación y footer Hover/Activo. Enlace activo o al pasar el ratón.
- `#9EB08C`: Background secundario (secciones destacada).
- `#6B4E37`: Texto sobre background claro, títulos.
- `#333333`: Texto sobre background claro.
- `#FFFFFF`: Texto sobre background oscuro. Texto principal sobre fondos oscuros.
- `#4D7B38`: Enlaces. Links inline en párrafos, títulos secundarios.
- `#C93F2D`: Enlace (hover). Color al pasar el ratón.
- `#4D7B38`: Botón principal. Fondo de botón
- `#FFFFFF`: Texto de botón principal.
- `#4D7B38`: Bordes de botones
- `#78A75A`: Botón principal (hover). Estado hover del botón.
- `Transparente`: Botón secundario fondo.
- `#4D7B38`: Botón secundario texto.
- `#4D7B38`: Botón secundario hover fondo.
- `#FFFFFF`: Botón secundario texto cuando hay hover.
